# OpenCV Visualizer Frontend

This is the frontend for the OpenCV Visualizer application, built using Next.js and Tailwind CSS.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view the application in your browser.
